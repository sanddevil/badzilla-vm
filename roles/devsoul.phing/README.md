Phing Ansible role
=========
[![Build Status](https://travis-ci.com/devsoul/ansible-role-phing.svg?branch=master)](https://travis-ci.com/devsoul/ansible-role-phing)

If you are a PHP developer and you use Phing.

Requirements
------------
  - `php > 5.2` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phing

Check if it's installed:
------
    phing --version
